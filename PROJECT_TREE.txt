PRIMEROS PASOS
==============

1️⃣  Lee la documentación:
    → SETUP.md        (Guía de instalación)
    → README.md       (Overview del proyecto)

2️⃣  Instala dependencias:
    → npm install
    → composer install

3️⃣  Configura el entorno:
    → cp .env.example .env
    → php artisan key:generate

4️⃣  Compila los assets:
    → npm run dev     (Terminal 1 - Con hot reload)

5️⃣  Inicia el servidor:
    → php artisan serve  (Terminal 2)

6️⃣  Abre en navegador:
    → http://localhost:8000
